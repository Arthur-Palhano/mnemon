# Mnemon

## Mnemon is a simple CLI notepad for basic terminal use

---

## Installation
> git clone https://github.com/Arthur-Palhano/mnemon.git

> cd mnemon

> main.exe

or
> go run main.go

## Tags
* n -> Create a new file
* r -> Print the file in terminal
* rn -> Print the file in notepad
* e -> open the file in notepad
* d -> delete a file
